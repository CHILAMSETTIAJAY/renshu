DDL

DDL statements are used to define, manage, and modify the structure of a database and its objects, such as tables, indexes, and constraints

DML
Data Manipulation Language

DML statements are used to manipulate data stored in a database. They perform operations like inserting, updating, and deleting records.

DQL
Data Query Languaget

DQL statements are used to retrieve data from a database. These statements are primarily focused on querying data and do not modify the data's structure.

DCL
Data Control Language

DCL statements are used to control access to data within a database. They manage user privileges and permissions.

TCL statements are used to manage transactions within a database. They ensure the integrity of data by allowing you to commit or roll back changes.
